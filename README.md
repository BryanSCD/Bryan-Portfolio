## Getting Started

Setting up your project locally.

### Prerequisites
  
* yarn
  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/BryanSCD/Bryan-Portfolio.git
   ```
2. Ininitialize yarn
   ```sh
   yarn
   ```
3. Run
   ```js
   yarn dev
   ```
